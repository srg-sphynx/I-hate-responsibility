🔒 Privacy Policy

Last updated: April 2026

I Hate Responsibility is a free, open-source web application that generates humorous blame excuses. We take your privacy seriously — which is ironic, given the app's theme.

1. Data We Collect

Short answer: Nothing.

We have no servers, no databases, no analytics, and no interest in your personal information. We set a single functional cookie (ihr-consent) to remember your consent choice — it is never used for tracking.

2. Local Storage

When you accept the consent banner, we store your excuse history in your browser's localStorage. This data:

  • Lives entirely on your device
  • Is never transmitted to any server
  • Can be deleted at any time using the "Clear All" button or your browser settings
  • Contains only the generated excuse text, category, and timestamp

If you decline the consent banner, no data is stored whatsoever.

3. Third-Party API

When you generate an excuse, a network request is sent to the Blame as a Service API. We do not control this API. Your IP address may be visible to their server as part of standard HTTP communication. Please refer to their repository for their policies.

4. Visitor Badge

The footer contains a visitor counter badge provided by visitorbadge.io. This is an external service that counts page visits. It does not collect personal data or use cookies. See their privacy policy for details.

5. CORS Proxy

If the API cannot be reached directly due to browser security restrictions (CORS), a proxy service (corsproxy.io) may be used to relay the request. This proxy only forwards the API request and response — no personal data is sent through it.

6. Your Rights

You have full control over your data at all times:

  • View: Your history is visible in the app's "Hall of Excuses"
  • Delete: Click "Clear All" or clear your browser's site data
  • Prevent: Decline the consent banner to opt out entirely

7. Changes

If this policy is updated, the "Last updated" date above will change. Since the app has no user accounts or contact information, we cannot notify you directly — but the source code and this page are always available on GitHub.
